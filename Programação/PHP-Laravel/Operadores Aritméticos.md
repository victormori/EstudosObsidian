- ==+== -> operador de adição
- ==-== -> operador de subtração
- ==* ==-> operador de multiplicação
- ==/== -> operador de divisão real
- ==%== -> operador de módulo 


#### Prioridades nas operações:

==A prioridade de execução do PHP== 

- parênteses ()
- multiplicação( * ), divisão( / ) e módulo.
- adição( + ) e subtração( - )


para priorizar o operações é utilizado os parênteses ==()== 

```php
$numero2 = 3;
$numero1 = 4;
echo "A média da soma dos números é: ". ($numero1 + $numero2) / 2; 
```

No exemplo acima ele prioriza a soma dos números pra depois dividir por 2;
