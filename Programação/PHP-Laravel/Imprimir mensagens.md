Para imprimir uma mensagem em PHP utilizamos ECHO

```php 
<?php ?>

echo "Olá, mundo" 
```

Como foi apresentado na [[Introdução]], também podemos usar o echo dentro de um código HTML, sendo possível usar h1, h2, h3, e outros códigos de formatação de texto dentro do código PHP.

## Concatenação:

Para realizar a concatenação no PHP é necessário utilizar ==.==

```php
$numero2 = 3;
$numero1 = 4;
$soma = $numero2 + $numero1;

echo "A soma dos números é: ". $numero1 + numero2;
echo "A soma dos números é: $soma";
```
